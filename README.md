# Imperium
A game by Windsdon.

Most of the action takes place on the client. The server mostly
broadcasts changes and initializes stuff.

# Running
Requires Node.
```
$ git clone git@github.com:Windsdon/imperium.git
$ cd imperium
$ npm install
$ node index
```
